## üöÄ Why Use This?
Every social media platform uses a different background color and circular crop. A photo that looks great on **Instagram Dark Mode** might lose its impact on **Gmail Light Mode**. This tool allows you to:
- **Test Environments:** Switch between Instagram, WhatsApp, Gmail, Facebook, LinkedIn, and X.
- **Toggle Themes:** Instantly swap between Dark and Light UI modes.
- **Precision Fitting:** Use sliders to adjust Zoom, Rotation, and X/Y positioning for the perfect crop.

- **Zero Server Uploads:** Your photos stay private. Processing is done entirely in your browser using the FileReader API.

- **Platform Specifics:** Background colors are matched to the official hex codes of each platform.
- **Responsive Design:** Works on both desktop and mobile browsers.
